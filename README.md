# MovieRecommendationSystem
[ML] Movie Recommendation System

## Usage
prepare_imdb_datasets.py - generates title.merged.sav file contains database:
```
 N   Column         Dtype  
---  ------         -----  
 0   tconst         object 
 1   primaryTitle   object 
 2   genres         object 
 3   averageRating  float64
 4   numVotes       int64  
 5   key_data       object  
```
